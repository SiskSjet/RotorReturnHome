This simple yet useful mod allows players to easily return their rotors to their home position with just the press of a terminal button.
No more manual adjustments or scripts required!

[h2]🛠︰Info ・─────────────・・・[/h2]
Adds a button to start return the rotor to home position and a slider to set the home position angle to rotor terminal controls.

❗❗❗ Currently only advanced rotors and hinges are supported because of a bug.
If you really want support for it please vote for this at least 3 years old [url=https://support.keenswh.com/spaceengineers/pc/topic/mod-api-terminal-controls-cannot-be-added-to-rotors]bug[/url]❗❗❗

[h2]🛠︰Programableblock API ・──・・[/h2]
you can also use this mod in your own scripts by using the following actions and properties

[h3]🛠︰Actions[/h3]
[list]
[*][b]ReturnToHome[/b]: [i]Return to home position[/i]
[*][b]ResetHomePositionAngle[/b]: [i]Reset home position angle[/i]
[*][b]IncreaseHomePositionAngle[/b]: [i]Increase home position angle[/i]
[*][b]DecreaseHomePositionAngle[/b]: [i]Decrease home position angle[/i]
[*][b]DisableRotorWhenHomeAngleReached[/b]: [i]Disable rotor when hp reached On/Off[/i]
[/list]

[h3]🛠︰Properties[/h3]
[list]
[*][b]HomePositionAngle[/b]: float
[*][b]DisableRotorWhenHomeAngleReached[/b]: bool
[/list]

Dieser einfache, aber nützliche Mod ermöglicht es Spielern, ihre Rotoren ganz einfach mit nur einem Tastendruck am Terminal in ihre Ausgangsposition zurückzubringen.
Keine manuellen Anpassungen oder Skripte mehr erforderlich!

[h2]🛠︰Info ・─────────────・・・[/h2]
Fügt eine Schaltfläche zum Starten der Rückkehr des Rotors in die Ausgangsposition und einen Schieberegler zum Festlegen des Ausgangspositionswinkels für die Steuerung des Rotorterminals hinzu.

❗❗❗ Aufgrund eines Bugs werden derzeit nur erweiterte Rotoren und Scharniere unterstützt.
Wenn ihr wirklich Unterstützung dafür wünschen, stimme bitte für diesen mindestens 3 Jahre alten Fehler [url=https://support.keenswh.com/spaceengineers/pc/topic/mod-api-terminal-controls-cannot-be-added-to-rotors]Bug[/url]❗❗❗

[h2]🛠︰Programableblock API ・──・・[/h2]
Ihr könnt diese Mod auch in eigenen Skripten verwenden, indem die folgenden Aktionen und Eigenschaften genutzt werden können

[h3]🛠︰Actions[/h3]
[list]
[*][b]ReturnToHome[/b]: [i]Zur Ausgangsposition zurückkehren[/i]
[*][b]ResetHomePositionAngle[/b]: [i]Ausgangspositionswinkel zurücksetzen[/i]
[*][b]IncreaseHomePositionAngle[/b]: [i]Ausgangspositionswinkel erhöhen[/i]
[*][b]DecreaseHomePositionAngle[/b]: [i]Ausgangspositionswinkel verringern[/i]
[*][b]DisableRotorWhenHomeAngleReached[/b]: [i]Deaktiviere den Rotor, wenn die Ausgangsposition erreicht wird An/Aus[/i]
[/list]

[h3]🛠︰Properties[/h3]
[list]
[*][b]HomePositionAngle[/b]: float
[*][b]DisableRotorWhenHomeAngleReached[/b]: bool
[/list]
